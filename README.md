# AI Credit Scoring ML Pipeline

End-to-End машинное обучение пайплайн для скоринга кредитных заявок. Проект демонстрирует полный цикл разработки ML-продукта: от разведочного анализа данных (EDA) и обучения модели до деплоя через FastAPI и создания пользовательского интерфейса на Streamlit.

## Архитектура проекта
1. **ML Pipeline (`src/train.py`)**: Обучение Logistic Regression с балансировкой классов и обязательным масштабированием признаков (`StandardScaler`).
2. **API Service (`src/service.py`)**: FastAPI микросервис для инференса модели с валидацией входных данных через Pydantic и логированием запросов.
3. **Frontend (`app.py`)**: Интерактивный веб-интерфейс на Streamlit для симуляции подачи заявки клиентом.
4. **EDA (`exploratory_analysis.ipynb`)**: Разведочный анализ данных, выявление дисбаланса классов и корреляций между признаками.

## О датасете
Проект использует данные, структурированные по принципу реальных кредитных скоринговых моделей (вдохновлено открытыми датасетами кредитного скоринга, включая данные Т-Банка). 

**Признаки (Features):**
- `age`: Возраст заемщика (числовой)
- `income`: Ежемесячный доход в тыс. рублей (числовой)
- `education`, `work`, `car`: Социальные и имущественные маркеры (бинарные)

**Целевая переменная (Target):**
- `default`: Факт дефолта (1) или успешного погашения (0).

**Особенности работы с данными:**
В реальных кредитных данных наблюдается сильный **дисбаланс классов** (дефолтов значительно меньше, чем успешных кредитов). Для борьбы с этим в модели используется параметр `class_weight="balanced"`, что автоматически увеличивает штраф за ошибку на миноритарном классе и повышает полноту (Recall) обнаружения рискованных заемщиков.

## Быстрый старт

### Локальный запуск
```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Обучить модель (создаст файлы в папке models/)
python src/train.py

# 3. Запустить API сервис (Swagger документация будет доступна на http://127.0.0.1:8000/docs)
uvicorn src.service:app --reload

# 4. В новом терминале запустить фронтенд
streamlit run app.py


Стек технологий
Machine Learning: Scikit-Learn (Logistic Regression, StandardScaler), Pandas, NumPy
Backend: FastAPI, Pydantic, Uvicorn
Frontend: Streamlit
Tools: Git, Jupyter Notebook

Метрики модели
Precision: ~0.75 (минимизация ложных одобрений)
Recall: ~0.70 (выявление реальных рисков)
Использован class_weight="balanced" для эффективной работы с дисбалансом классов.
